Heqet

Isaac Reilly's senior project in Computer Science at Yale University

Heqet can convert music from Euterpea, manipulate it, and generate code for displaying it as a printed score using Lilypond. If you want to use Heqet, clone this repository (on the master branch) and run "cabal build". Then check out my tutorial! If you're here to grade my project, here's my report.
